## Publicar en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos de esta carpeta al repositorio.
3. Entrar a Settings > Pages.
4. En Source, seleccionar Deploy from a branch.
5. Elegir la rama main y carpeta /root.
6. Guardar y esperar a que GitHub genere el enlace.

## Nota de seguridad

Este login protege la vista para una presentacion, pero GitHub Pages no es un sistema privado de autenticacion real. Para uso formal con datos sensibles, se recomienda SharePoint autorizado por IT, servidor interno o autenticacion corporativa.
